
# WebScraper3000

## Description

This scraper parses books.toscrape.com and websites with a similar structure to extract book titles and prices and save them to a CSV file.

## Installation

### Prerequisites

- Python 3.13.5

### Steps

1. Clone the repository
2. Create a virtual environment: python3 -m venv .venv
3. Activate it: source .venv/bin/activate
4. Install dependencies: pip install -r requirements.txt
5. Run the application: python3 web-scraper-3000.py

## Usage

Type any website url into the search box and specify what items you wish to parse for. ***Note that some websites do not allow scrapers to run.***
