# WebScraper3000

## Description
This project allows users to parse and extract select data from websites quickly and effectively. 

## Installation
1. Clone the repository.
2. Run 'apt install'.
3. Run 'WebScraper3000'.

## Usage
Type any website url into the search box and specify what items you wish to parse for. ***Note that some websites do not allow scrapers to run.***
